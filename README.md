# 🚀 SME AI Integration Hub & Readiness Scorecard

**Live Web Application:** [https://sme-ai-adoption-platform.streamlit.app/]

## 📌 Project Overview
The **SME AI Integration Hub** is a cloud-hosted, zero-code diagnostic web application designed to help Small and Medium Enterprises (SMEs) safely transition to AI-driven workflows. 

This software is the practical implementation of my final-year Research Honours project, which identified that SMEs are blocked from AI adoption by a **60% technical skill gap** and a **64% "Trust-Accuracy Deficit."** Instead of offering generic advice, this logic engine allows business owners to self-diagnose their digital maturity and generates a custom, risk-mitigated adoption roadmap.

## ✨ Key Features
* **Interactive Readiness Scorecard:** Evaluates users based on their current digital infrastructure, in-house technical capability, and risk tolerance.
* **Deterministic Logic Engine:** Processes user inputs to generate a quantified "Readiness Score."
* **Custom 5-Phase Roadmap:** Automatically places the business into a structured adoption framework (ranging from basic data digitization to a Phase-4 "Human-in-the-Loop" pilot).
* **Zero-Code Interface:** Built with a highly intuitive Streamlit frontend, ensuring accessibility for non-technical founders.

## 🛠️ Tech Stack & Architecture
* **Frontend/UI:** Streamlit
* **Backend Logic:** Python 3.9+
* **Data Processing:** Pandas, NumPy
* **Deployment:** Streamlit Community Cloud (Serverless)
